Versão 2.1: A Evolução Final

Focada na otimização para ambientes internos, a Versão 2.1 do B.A.R.T. teve 70% de sua estrutura fabricada via impressão 3D. SENSORY SUITE

Captação de Dados Multimodal

MODULE_A: REALSENSE_D435

Intel RealSense

Capacidades RGB-D para percepção de profundidade estereoscópica. Gera nuvens de pontos (Point Clouds) densas para detecção de obstáculos e reconhecimento de objetos em 3D.

MODULE_B: RPLIDAR_C1

RPLidar C1

Escaneamento laser 360° de alta frequência. Fornece o esqueleto geométrico preciso do ambiente, essencial para o mapeamento de longo alcance e navegação planar estável.

A Magia da Fusão

RTAB-Map: Gestão de Memória

STM Short-Term Memory para processamento imediato.
WM Working Memory para loop closure detection.
LTM Long-Term Memory para persistência de mapas globais.

O grande diferencial do B.A.R.T. é como o LiDAR ICP (Iterative Closest Point) atua como uma rede de segurança. Em ambientes brancos ou sem textura (featureless), onde a odometria visual da RealSense falha catastróficamente, os dados do laser mantêm o robô localizado no grid, prevenindo a perda total do estado de navegação.
